# Windows Terminal Setup

- Enable Acrylic Effect.
- Set acrylic opacity to 70.
- Use Blazer Color scheme from [WindowsTerminalThemes.dev](https://windowsterminalthemes.dev)
	- To install, add the theme's settings object to the `schemes` array in `settings.json` of the Windows Terminal. Refer the docs [here](https://docs.microsoft.com/en-gb/windows/terminal/customize-settings/color-schemes)
- Disable `Copy on Select`
- Use Cascaida Code's Nerd Font Variant -- [Caskaydia Cove NF](https://www.nerdfonts.com/font-downloads)
- Setup [Oh My Posh](https://ohmyposh.dev/)
- Use these `Oh My Posh` themes from the official repo:
	- [Bubbles](https://github.com/JanDeDobbeleer/oh-my-posh/blob/main/themes/bubbles.omp.json)
	- [M365Princess](https://github.com/JanDeDobbeleer/oh-my-posh/blob/main/themes/M365Princess.omp.json)
	- Or use [my custom themes](https://github.com/icy-comet/oh-my-posh-themes)
