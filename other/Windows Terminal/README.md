# Windows Terminal Setup

- Enable Acrylic Effect.
- Use Acrylic Effect in Tab Row as well.
- Set acrylic opacity to 80.
- Disable `Copy on Select`
- Use Cascadia Code NerdFont variant or custom-patched MonoLisa's.
- Change WSL profiles' default starting point to distro's home.

### Color Schemes
To install, add the theme's JSON to the `schemes` array in `settings.json` of the Windows Terminal.
Refer the docs [here](https://docs.microsoft.com/en-gb/windows/terminal/customize-settings/color-schemes)

- [WindowsTerminalThemes.dev](https://windowsterminalthemes.dev)
  - Blazer color scheme
  - Overnight Slumber scheme 
- [Catppuccin Theme](https://github.com/catppuccin/windows-terminal)
